## 1. ログイン失敗確認

```bash
TARGET_URL=http://localhost:8081 \
USERNAME=k6user01 \
BAD_PASSWORD=wrong-password \
EXPECTED_STATUS=401 \
k6 run CT1001_AUTH_LOGIN_INVALID_PASSWORD.js


## 2. 未ログイン状態確認

```bash
TARGET_URL=http://localhost:8081 \
STATUS_PATH=/api/auth/status \
EXPECTED_STATUS=401 \
k6 run CT1001_AUTH_STATUS_UNAUTHENTICATED.js


## 3. ログイン → ログアウト → ステータス確認

```bash
TARGET_URL=http://localhost:8081 \
USERNAME=k6user01 \
PASSWORD=password123 \
STATUS_PATH=/api/auth/status \
LOGOUT_PATH=/api/auth/logout \
EXPECTED_STATUS_BEFORE_LOGOUT=200 \
EXPECTED_LOGOUT_STATUS=200 \
EXPECTED_STATUS_AFTER_LOGOUT=401 \
K6_VUS=1 \
K6_DURATION=5s \
k6 run CT1003_AUTH_LOGOUT_THEN_STATUS.js
