# 📘 `generate-authors-from-multiple-repos.ps1` 説明書

## 📌 概要

このスクリプトは、指定した SVN リポジトリの複数サブディレクトリ（モジュール）に対してログを走査し、すべてのコミッター（ユーザー名）を抽出・統合し、Git 移行時に使用する `authors.txt` 形式で出力します。

## 🧩 使い方

### ✅ 実行コマンド例

```powershell
.\generate-authors-from-multiple-repos.ps1 `
    -baseUrl "http://sys.dev.genesys-eco.jp/svn/snp2" `
    -subdirs @("documents", "kun", "navi", "sanpaikun_cron") `
    -outputFile "authors.txt"
```

### 📝 パラメータ説明

| パラメータ名       | 説明                                                |
| ------------ | ------------------------------------------------- |
| `baseUrl`    | SVN リポジトリのベース URL。`/trunk` や `/branches` などは含めない。 |
| `subdirs`    | SVN リポジトリのサブディレクトリ（モジュール名）一覧。                     |
| `outputFile` | 出力ファイル名。デフォルトは `authors.txt`。                     |

---

## 🛠 スクリプトの処理概要

### 1. 各サブディレクトリのログ取得

```powershell
$log = svn log $url --quiet
```

* `svn log` コマンドを使って、リビジョン履歴（著者情報含む）を取得
* `--quiet` によりログのヘッダのみを対象（無駄な本文は省略）

### 2. ユーザー名の抽出と統合

```powershell
$authors = $log | Where-Object { $_ -match "^r\d+" } | ForEach-Object {
    ($_ -split '\|')[1].Trim()
}
```

* `r12345 | username | ...` の形式から `username` 部分を抽出
* 各リポジトリでの全ユーザー名を `$allAuthors` に追加

### 3. 重複除去とファイル出力

```powershell
$uniqueAuthors = $allAuthors | Sort-Object -Unique
```

```powershell
"$username = $username <$username@example.com>"
```

* 重複を除いたユーザーリストを `authors.txt` 形式で整形
* 仮のメールアドレス（`@example.com`）は後で手動で修正可能

---

## 📄 出力ファイルフォーマット

```
yamada = yamada <yamada@example.com>
suzuki = suzuki <suzuki@example.com>
tanaka = tanaka <tanaka@example.com>
```

この形式は、`git svn` コマンドの `--authors-file` オプションで使用可能です。

---

## ✅ 利用上の注意点

* **`svn` CLI ツールが必要**です。コマンドが通る状態で実行してください。
* SVN アクセスに認証が必要な場合は、事前に `.subversion/servers` や `.netrc` に設定してください。
* 仮メールアドレスは、後で本物のものに編集してください。
