# FileSlot モジュール仕様書

## 1. モジュール概要

### 1-1. 目的
`FileSlot` は、1件分のファイル名表示・クリックダウンロード・削除ボタン表示を担うファイル添付欄の最小単位コンポーネントである。

### 1-2. 適用範囲
- 複数添付 UI の各スロット表示
- ファイル名クリックによるダウンロード導線
- 添付済み/未添付のプレースホルダー表示

---

## 2. 設計方針

### 2-1. アーキテクチャ
- 文字列表示には `ClickableFont14` を利用する。
- 削除操作には `IconButtonBase` と `CloseIcon` を利用する。
- ファイル未設定時はプレースホルダー文字列を表示する。

### 2-2. 統一ルール
- `disabled` 時は背景色・文字色を変更し、削除ボタンは非表示とする。
- ファイル名は 1 行省略表示とする。
- ダウンロードクリック導線はファイル名部分に集約する。

## 4. コンポーネント仕様

**主な props:**
- `file?: UploadedFile` - 表示対象ファイル
- `onClick: () => void` - クリック時処理
- `onDelete?: () => void` - 削除時処理
- `disabled?: boolean` - 非活性制御

**仕様:**
- `file` がある場合は `file.fileName` を表示する。
- `file` がない場合は `------------------------------` を表示する。
- `file && onDelete && !disabled` のときのみ削除ボタンを表示する。
- 現行実装では `disabled` 時でも `onClick` 自体は実行される。
