# 帳票出力_テーブル定義書

## 1. 目的
帳票出力で利用する主要テーブルを定義する。

## 2. 帳票定義テーブル

### 2-1. `report_master`
| カラム | 型 | 制約 | 説明 |
| --- | --- | --- | --- |
| `report_id` | VARCHAR(50) | PK, NOT NULL | 帳票ID |
| `report_name` | VARCHAR(100) | NOT NULL | 帳票名 |
| `template_file` | VARCHAR(255) | NOT NULL | テンプレートパス |
| `output_format` | INT | NOT NULL | 1=PDF, 2=EXCEL |
| `description` | VARCHAR(500) | NULL | 説明 |
| `updated_at` | DATETIME | NOT NULL | 更新日時 |
| `updated_by` | VARCHAR(50) | NOT NULL | 更新者 |

### 2-2. `report_layout`
| カラム | 型 | 制約 | 説明 |
| --- | --- | --- | --- |
| `report_id` | VARCHAR(50) | PK, FK, NOT NULL | 帳票ID |
| `column_id` | VARCHAR(50) | PK, NOT NULL | 項目ID |
| `table_name` | VARCHAR(100) | NOT NULL | テーブル名 |
| `column_name` | VARCHAR(100) | NOT NULL | カラム名 |
| `display_label` | VARCHAR(100) | NOT NULL | 表示名 |
| `data_type` | INT | NOT NULL | 1=文字列,2=数値,3=日付 |
| `display_order` | INT | NOT NULL | 表示順 |
| `visible_flag` | INT | NOT NULL | 1=表示,0=非表示 |
| `format_pattern` | VARCHAR(50) | NULL | 書式 |
| `required_flag` | INT | NOT NULL | 1=必須,0=任意 |
| `default_value` | VARCHAR(100) | NULL | 初期値 |
| `remarks` | VARCHAR(500) | NULL | 備考 |
| `updated_at` | DATETIME | NOT NULL | 更新日時 |
| `updated_by` | VARCHAR(50) | NOT NULL | 更新者 |

## 3. 非同期状態テーブル
- 帳票の非同期ジョブ状態は `async_job_execution` を利用する。
- 詳細定義は `async_job_execution定義書.md` を参照。
