# 02_設定値テンプレート

---

## 1. 目的
本資料は、アプリケーションで使用される設定値を整理し、
**設定キー・用途・参照箇所を明確化すること**を目的とする。

---

## 2. 適用範囲
- 対象アプリ：{appserver / batchserver / servercommon / frontend}
- 対象設定：
  - application.yml
  - application-*.yml
  - 環境変数
  - 外部設定（S3 / DB / API など）

---

## 3. 基本方針

- 設定値は「キー単位」で管理する
- 実装コードを正とする
- 未使用設定は明記する
- 設定の責務（どこで使われるか）を明確にする
- 推測は禁止。不明は「要確認」とする

## 7. 注意事項

- 設定キーは実装コードと完全一致させること
- 廃止予定の設定は削除せず「未使用」として明記すること
- 環境依存値（パスワード等）は値ではなくキーのみ記載すること

## 記載ルール

- 設定キー中心で記載すること
- 実装コードは記載しない
- フローは記載しない
- DB定義は記載しない
- 設定の「意味」と「参照先」のみを明確にする
