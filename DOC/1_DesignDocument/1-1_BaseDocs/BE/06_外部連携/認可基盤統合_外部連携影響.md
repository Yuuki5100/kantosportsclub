# 認可基盤統合_外部連携影響

## 1. 目的
認可基盤統合（2026-04-02実装）が外部連携I/Fに与える影響有無を明確化する。

## 2. 結論
- 外部API仕様の変更はなし
- 外部連携先へのリクエスト/レスポンス項目追加なし
- URL/ヘッダ/署名仕様の変更なし

## 3. 影響範囲
- 変更は appserver 内部の認可判定経路と設定値に限定
- 正本テーブル参照（`endpoint_authority_mapping`, `role_permission`）は内部DBアクセスのみ

## 4. 非影響項目
- sync連携
- WebSocket連携
- 帳票出力連携
- 認証外部連携（internal/gbiz）の通信仕様
