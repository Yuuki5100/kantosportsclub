## 2. 要確認事項まとめ
- 認証導線（callback / OIDCテスト）は整理設計後に対応する。
- Stories 内の `alert` は対象外。

## 3. 次アクション候補（未実施）
- 認証導線は可能な限り `authService` / `useAuth` へ寄せる。
- 寄せられない場合は共通コンポーネント化し、再利用方針へ追加・運用ルール整備。
